Save and Restore
This function allow user to save and restore the analysis with data and status. This function is also useful to send the data to other.
  Click "Save analysis to restore file" button to download the analysis data file.
  Store the data file in your PC.
The file name can be changed, but do not change the file suffix (.qma).
  Upload the data file from home page of QUMA to restore the analysis.
  The analysis will restore.

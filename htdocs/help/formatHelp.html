Graphics formats
QUMA supports three file formats for graphics data download.
  PNG (default): Portable Network Graphics is a popular bitmapped image format and can be opened by many applications. PNG file format is easy to use, however, larger file size is required for high-resolution figure.
  SVG: Scalable Vector Graphics is a file format for vector graphics and can be edited by Adobe Illustrator and other vector graphics applications. SVG file format is suitable to use for a journal figure, because high-quality and editable.
  SVGZ: compressed SVG format. The size of SVGZ file is smaller than standard SVG file (about 20%).

Multiple alignment Help
Download Format
Plain text (.txt)
Human readable text format.
Multi-FASTA (.fa)
Multi-FASTA format file can be used as an input file of other programs (such as CyMATE: useful tool for both CpG and non-CpG methylation analysis).
Tab separate values (.tsv)
Comma separated values (.csv)
These formats can be opened by spreadsheet softwares.
Example
C Methylated C of CpG site
T Unmethylated C of CpG site
C Unconverted C of non CpG site
T Mismatch or gap
C Not CpG site at

Target genomic sequence
Select the genomic sequence file of the target region to upload. The genomic sequence must be an unconverted sequence between PCR primer pair (it is not necessary to convert "C" to "T").
Plain sequence, FASTA or GenBank format is acceptable.
Only rich text format (with ".rtf" file extension) or plain text format text files are acceptable for file upload. Binary files (such as Microsoft Word files) will not work.
Rich text format files can be created with TextEdit (Macintosh), WordPad (Windows) and many word processors. Plain text files can be created with TextEdit (Macintosh), NotePad (Windows), and many word processors or text editors.

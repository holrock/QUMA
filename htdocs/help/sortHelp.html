Sorting conditions
Order of bisulfite sequences can be changed by several parameters and ascending/descending order. Click "Renew" button to reflect the changed conditions. (CpH: CpA, CpC, CpT)
  user specified order: the value of "order" column
  number of methylated CpGs
  number of unconverted CpHs
  percent converted CpHs: percent value of "converted CpHs" / "total CpHs"
  number of mismatches: number of alignment mismatch (includes gap)
  percent identity: percent value of alignment homology
  sequence name: the value of "Sequence name" column

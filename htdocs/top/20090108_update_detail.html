Detailed information about the update at Jan 8 2009
  "Save and Restore" function now available. This function allow user to save the data with status and condition of the analysis to a file. This function is also useful to send the data to other.
  Multiple-alignment format sequence display is useful for analysis of allele specific sequence differences (SNPs) of imprinting or clinical samples. Multi-FASTA format download file can be used as an input file of other programs (such as CyMATE: useful tool for both CpG and non-CpG methylation analysis).
  Now support circle graph style methylation pattern figure. This type of figure is useful for comparing many types of samples.
  Now support CpG number display function at the methylation pattern figures.
  Now display standard deviation (S.D.) and standard error (S.E.) for the CpG methylation status (between CpG sites and between bisulfite sequences).

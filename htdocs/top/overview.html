How to use the statistical analysis
QUMA is a web-based tool for CpG methylation analysis.
You can easily align, visualize and quantify bisulfite sequence data!
 QUMA User's Manual (pdf file: 6.0MB)
QUMA consists of two separate analyses; a methylation status analysis mode using one group of bisulfite sequences and a statistical analysis mode using two groups of bisulfite sequences.
